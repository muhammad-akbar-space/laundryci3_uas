Laundry Asrama Mahasiswa (CI3)

Aplikasi ini merupakan sistem transaksi laundry asrama
khusus mahasiswa yang dibuat untuk memenuhi Tugas UAS.

Fitur Utama:
- Input transaksi laundry mahasiswa
- Edit dan hapus data transaksi
- Notifikasi berhasil tambah, ubah, dan hapus data
- Tampilan sederhana dan mudah digunakan

Teknologi yang Digunakan:
- PHP
- CodeIgniter 3
- Bootstrap
- MySQL

Catatan:
Aplikasi dijalankan secara lokal menggunakan XAMPP.
